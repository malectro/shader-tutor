# VIM Instructor

## Premise
We're building a web code editor that runs in a VIM mode. When the user uses their mouse,
the editor will share the action with Claude and ask it to suggest a keyboard command that
would have worked better.

The idea is to teach people how to use VIM without slowing them down.

## Technology
We will probably use CodeMirror for this.

## Instructions
Please make a plan to build out the basics of a CodeMirror editor webapp running in a
solid VIM mode. We should allow this app to make calls to the Claude API and display
these hints in a side panel.
